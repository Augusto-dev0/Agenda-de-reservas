# 📅 Agenda de Reservas - Espaço de Lazer

- Sistema web simples e moderno para gerenciamento de reservas de um espaço de lazer.
- Projeto desenvolvido com HTML, CSS e JavaScript puro, utilizando localStorage para armazenamento dos dados no navegador.

# 🚀 Funcionalidades:
ㅤ
- Cadastro de novas reservas
- Edição de reservas
- Exclusão de reservas
- Bloqueio de datas duplicadas
- Status da reserva (Pendente, Confirmado, Finalizado)
- Status de pagamento com badge colorido (Pago / Pendente)
- Visualização de observação ao clicar no botão 👁
- Painel financeiro automático
- Filtro por mês
- Dark Mode 🌙
- Layout 100% responsivo (Celular, Tablet e Desktop)
- Dados salvos no navegador
- Exportar em pdf a reserva.

# 🖥️ Tecnologias Utilizadas:
- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Font Awesome (ícones)
-LocalStorage

# 📊 Painel Financeiro:
- O sistema calcula automaticamente:
- 💰 Total Recebido
- 💸 Total Pendente
- 📅 Total de Reservas
Tudo atualizado em tempo real.

# 📱 Responsividade:
O sistema foi desenvolvido para funcionar perfeitamente em:
- 📱 Celulares
- 📲 Tablets
- 💻 Notebooks
- 🖥 Computadores
 - A tabela se adapta e permite rolagem horizontal em telas menores.

# 💾 Armazenamento:
- Os dados são armazenados no localStorage do navegador. Isso significa:
- Não precisa de banco de dados
- Não precisa de backend
